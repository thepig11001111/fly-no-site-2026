# FLY-NO Website

FLY-NO static website prototype (HTML/CSS/JS) — a peer-to-peer waste marketplace preventing fly-tipping through circular economy principles.

## Pages
- **Home** (`index.html`) — Hero, How It Works, Problem Statement, Impact Stats
- **Sell** (`sell.html`) — Seller portal with listing form, instant valuation, scheduling
- **Buy** (`buy.html`) — Buyer marketplace with filters, product grid, price trends
- **Drive** (`drive.html`) — Driver app with jobs board, active job, earnings

## Tech Stack
- HTML/CSS/JS (static)
- Chart.js for price trend graphs
- Google Fonts (Outfit + Inter)
